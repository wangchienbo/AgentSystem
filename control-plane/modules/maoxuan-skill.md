# maoxuan-skill

## Purpose
Integrate the Mao Zedong thinking framework (毛选.skill) as a built-in system skill for strategic analysis and problem diagnosis.

## Responsibilities
- provide LLM-powered analysis using Mao Xuan mental models
- serve as the strategic thinking / problem diagnosis capability
- integrate with the model interaction layer for deep analytical queries

## Reference source
- https://github.com/leezythu/maoxuan-skill

## Core mental models
- 矛盾分析法, 实践认识循环, 持久战略, 农村包围城市
- 统一战线, 群众路线, 纸老虎论
- 10 decision heuristics from Mao Xuan
