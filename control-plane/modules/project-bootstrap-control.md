# project-bootstrap-control

## Purpose
Bootstrap and maintain the initial project control layer for AgentSystem until finer-grained project, app, or module scopes are explicitly delegated by the master control skill.

## Responsibilities
- maintain the project anchor and core control-plane artifacts
- keep project-domain scope records coherent
- support project-level routing and structural governance
- record when more specialized subordinate scopes become stable enough to externalize

## Current status
- status: active
- owner: agent
- current focus: establish clean project-level master control routing for AgentSystem

## Notes
This is an initial bootstrap scope, not a claim that the entire repository should remain permanently governed as one undifferentiated unit.
