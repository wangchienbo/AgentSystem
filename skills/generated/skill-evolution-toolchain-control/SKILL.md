# Skill Evolution Toolchain Control

Structural subordinate skill for the `skill-evolution-toolchain` domain in AgentSystem.

## Scope
Owns focused work related to experience capture, suggestion flows, generated skill paths, executable skill handling, and reusable capability growth mechanisms.

## Entry conditions
Use this skill when the requested work is primarily inside generated-skill, suggestion, experience, or executable-skill evolution paths.
Escalate back to `agentsystem-master-control` when changes substantially impact project-wide boundaries, governance policy, or core platform contracts.

## Owned focus
- app/services/system_skills
- generated skill / executable skill / suggestion related services and flows
- experience-to-skill and capability-growth surfaces

## Operating rules
- prefer local toolchain context first
- escalate when work spills into broad telemetry governance or platform-core contract redesign
- update project control artifacts when toolchain boundaries or dependencies materially shift
