# 角色对话生成技能

## 核心能力
- **双角色对话**：两个角色之间的自然对话
- **多角色对话**：三人及以上群聊场景
- **内心独白**：角色内心活动的独白段落
- **场景性对话**：嵌入环境描写的完整对话场景

## 操作方式
- 角色对话生成：`call_asset_method(asset:novel_studio:v1, character_dialogue, {"novel_id": "xxx", "char1": "...", "char2": "...", "topic": "..."})`
- 需要先通过 `get_novel` 查看角色数据，了解角色性格作为对话基础

## 设计原则
1. **人各有声**：每个角色的语言风格、用词、句式、语气应与性格一致
2. **潜台词**：角色话里有话，说的和想的不一样，让读者揣摩
3. **展示关系**：对话应体现角色间的关系——亲疏、地位、情感
4. **推进情节**：对话应提供信息或推动情节，不做无意义闲聊
5. **避免信息 dump**：不要让角色说出他们已知的事实（"如你所知..."）——这是偷懒的 exposition
6. **动作穿插**：对话中插入角色动作、表情、环境，避免"头碰头"对话

## 对话格式示例
```
「你真的要去？」她盯着窗外的雨幕，没有回头。
「非去不可。」他把钥匙揣进口袋。
「哪怕回不来？」
沉默了一拍。「哪怕回不来。」
```
