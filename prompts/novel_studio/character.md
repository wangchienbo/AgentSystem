# 角色管理技能

## 核心能力
- **创建角色**：设定姓名、类型（主角/配角/反派/导师等）、性格标签（3-5个关键词）、背景故事、说话风格
- **修改角色**：更新已有角色的任意字段
- **删除角色**：从小说中移除角色
- **角色关系**：定义角色之间的人际关系（盟友/敌对/恋人/师徒等）
- **角色弧光**：规划角色的成长弧线（从→到）

## 操作方式
创建/修改/删除角色时，调用 `call_asset_method`：
- asset_id: `asset:novel_studio:v1`
- method: `add_character` / `update_character` / `delete_character`
- params: 包含 novel_id 和角色字段

查看当前所有角色：调用 `call_asset_method(asset_id="asset:novel_studio:v1", method="get_novel", params={"novel_id": "xxx"})` 从返回数据中获取 characters。

## 设计原则
1. **角色要有驱动力**：每个角色应该有欲望→动机→行动链
2. **角色要有弱点**：完美角色缺乏张力，赋予缺陷和内心冲突
3. **角色要有变化**：经历事件后角色应有所成长或改变
4. **关系要有张力**：角色关系是推动情节的核心引擎
5. **对话要差异化**：每个角色的说话风格、用词习惯应不同
